History

  1. 1995년 넷스케이프 '라이브 스크립트'
  2. 자바의 유명세를 본 따 '자바 스크립트'로 변경3. MS에서 Jscript발표
  3. ECMA에서 자바스크립트 표준 관리
  4. ECMA-262 표준 제정 -> ECMAScript
  5. ES5
  6. ECMAScript 6, ECMAScript 2015
